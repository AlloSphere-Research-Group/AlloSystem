Source

Bleeding edge development of AlloCore uses an internal subversion repository. Snapshots will be posted here.

Structure

The AlloCore source tree has the following structure: 
build/
	- location of built libraries

include/
	- contains all header files organized by module name

projects/
	- project files for building libraries and examples

src/
	- library source files (.c, .cpp)
	
dev/
	- external dependencies

unitTests/
	- unit tests for library

examples/
	- example projects using library
