AlloCore

About

AlloCore is a cross-platform suite of C++ components for building interactive multimedia tools and applications. In short, it provides a unified collection of easy-to-use classes and functions for system run-time control, media-related I/O interfaces and protocols, mathematical objects, graphics rendering, and spatial audio.
