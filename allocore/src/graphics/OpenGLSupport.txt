Not supported under OpenGL ES 2.0:
  - The type double and all double-precision commands (uses only the suffixes ’f ’, and ’i’).
  - Pre-defined vertex data arrays such as vertex, color, normal, texture coord arrays
  - The commands for directly specifying rectangles
  - The fixed function transformation pipeline
  - The fixed function lighting model
  - Anti-aliased lines and line stippling
  - Supports aliased point sprites only. The POINT_SPRITE default state is always TRUE.
  - Bitmap images
  - Feedback 
  - Display lists
