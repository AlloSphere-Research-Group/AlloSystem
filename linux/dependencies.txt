Installation Using apt-get
--------------------------
First resynchronize the package index files from their sources by running

	sudo apt-get update

Then for each package listed run

	sudo apt-get install <library_name>

or install all packages at once with

	sudo apt-get install libapr1-dev libglew-dev libglut3-dev portaudio19-dev libjack-dev freeimage-dev
